voipi

Give your apps, CLIs, and agents a voice. Universal, zero-dependency, free text-to-speech for JavaScript.

CLI

# Speak text (auto-selects best provider)
$ npx voipi "The quick brown fox jumps over the lazy dog"

# Choose a specific voice and speed
$ npx voipi "Hi" -v en-US-BrianNeural -r 1.5

# Save to file instead of playing
$ npx voipi speak "Hi" -o hello.mp3

# Use a specific provider
$ npx voipi speak "Bonjour" -p google-tts -v fr

# List available voices
$ npx voipi voices

Programmatic Usage

import { VoiPi } from "voipi";

const voice = new VoiPi();

// Speak text
await voice.speak("Hello world!");

// With a prioritized voice list (first available wins)
await voice.speak("Hello!", { voice: ["Samantha", "en-US-AriaNeural"], rate: 1.5 });

// Save to file
await voice.save("Hello!", "output.mp3");

// Get audio data with duration
const audio = await voice.toAudio("Hello world!");
console.log(`Duration: ${audio.duration}s`);

Providers

macOS

Native · Offline

Uses the native say command. Instant, no downloads. macOS only.

Edge TTS

Online · 322+ voices

Microsoft Edge's neural speech service. High-quality voices with rate, pitch, and volume control.

Google TTS

Online · 55+ languages

Google Translate's speech endpoint. Zero config, wide language support.

Piper

Local Neural · 40+ languages

Offline neural TTS. Auto-downloads models from HuggingFace. Fully offline after first run.

Browser TTS

Web Speech API

Built-in browser synthesis. No audio file output — speaks directly. Varies by browser/OS.

Auto (VoiPi)

Default · Fallback chain

Automatically picks the best provider: macOS → Piper → Edge TTS → Google TTS.
